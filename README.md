# Moonstone

A **fast** and **simple** algorithm checker for Windows, macOS and Linux!

## Installation

1. Run

```sh
cargo build --release
```

2. Add the executable from `target/release` to your PATH

## File structure

```
📁 tests
  ├── 📄 <name>.test
  └── ...
📄 main.cpp
📄 brute.cpp
📄 gen.cpp
```
